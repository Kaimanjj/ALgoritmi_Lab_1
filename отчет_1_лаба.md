3. Бинарный поиск  
Алгоритмическая сложность: O(log n)
   ```python
   def Binary_find(massiv, target):
    sorted_massiv = sorted(massiv)
    left,right = 0, len(sorted_massiv)-1
    while left <= right:
        mid = (right + 1)//2
        if target > sorted_massiv[mid]:
            left = mid + 1
        elif target < sorted_massiv[mid]:
            right = mid - 1
        else:
            return True
    return False
    ```

5. Сортировка выбором  
   Алгоритмическая сложность: O(n**2)  
   Пространственная сложность: O(n)
   ```python
   def Selection_sort(massiv):
    massiv = massiv.copy()
    n = len(massiv)
    for i in range(n):
        min_index = i
        for j in range(i + 1, n):
            if massiv[j] < massiv[min_index]:
                min_index = j
        massiv[i], massiv[min_index] = massiv[min_index], massiv[i] 
    return massiv
    ```
