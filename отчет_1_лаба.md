   def Find_max(massiv):
    max_1 = max_2 = massiv[0]
    if len(massiv)==1:
        return massiv[0]
    if len(massiv)>2:
        for i in massiv:
            if i>max_1:
                max_1 = i
                max_2 = max_1

    return max(massiv[0], massiv[1])

3. Бинарный поиск
   Алгоритмическая сложность: O(log n)
   ```python
   def Binary_find(massiv, target):
    sorted_massiv = sorted(massiv)
    left,right = 0, len(sorted_massiv)-1
    while left < right:
        mid = (right + 1)//2
        if target > sorted_massiv[mid]:
            left = mid + 1
        elif target < sorted_massiv[mid]:
            right = mid - 1
        else:
            return True
    return False
    ```
